# Project Description

## English

Cognilattice is a local-first MCP memory server for AI agents. It converts conversations, updates, decisions, tasks, code structure, and professional reasoning profiles into durable context that can be recalled across sessions. The system keeps protected top-level memories for project core knowledge, the latest conversation summary, and the current project summary, while SQLite, LanceDB, optional knowledge-graph integrations, and profile-aware scoring support long-term retrieval, curation, and stress-tested operation.

Short description for GitHub:

```text
Local-first MCP memory lattice for AI agents with curated summaries, profiles, graph context, and stress-tested retrieval.
```

## 中文

Cognilattice 是一个面向 AI Agent 的本地优先 MCP 记忆服务器。它把对话、项目更新、决策、任务、代码结构和专业化认知画像转化为可长期保留、可检索、可持续整理的上下文。系统会固定维护项目核心认知、最新对话总结和当前项目总结三类顶层记忆，并通过 SQLite、LanceDB、可选知识图谱集成和画像感知评分机制，支持跨会话唤醒、记忆策展、语义检索和经过压力测试的高负载运行。

GitHub 简介：

```text
本地优先的 AI Agent MCP 记忆网络，支持策展摘要、认知画像、图谱上下文与压力测试检索。
```
